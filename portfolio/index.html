Quiz Millionarie

GAME|GAMEPLAY

Mobile quiz game developed for Android and iOS using Unity3D and C#. I scripted game mechanics, UI animations, question importer, advertisements powered by Google Admob, and Google Play achievements.

wizards and Warlocks

GAME

Developed in C++ from scratch. It was engineered to work on the lowest end computers. It uses own Physics, Net and Graphics engines(Travelers Ocular Engine). Wizards & Warlocks is a team-based 3d-shooter local-network multiplayer which goal is defend or conquer the holy grail.

King of the Jungle

GAME

Developed for GNU/Linux platform, using C++, SFML graphics library and Box2D physics library. King of the Jungle is a local multiplayer 2D shooter. Matches are up to 4 players and AI can replace players if wanted. There are 2 different game modes: All vs All and Coin Collect.

RollaBlaze

GAME

Made for CPCRetroDev 2017 contest. Purely developed in Z80 assembly using CPCtelera library. This game is playable on a real Amstrad CPC 464 (1984). Rollablaze is a game whose goal is getting to the end of the level by dodging random obstacles while speed and amount of them are increasing.

Packedtory

GAME

Made as a personal challenge and senior thesis. Purely developed in assembly for Game Boy (also known as gbZ80). This game is playable on a clasic Game Boy, and is compatible with any posterior Game Boy (Game Boy Color and Game Boy Advance). Packedtory is a factory planning game where you have to create your own logistic station dealing with limited space and route boxes from entry to exit points.
